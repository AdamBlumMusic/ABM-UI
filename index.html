Code

Code can be written either inline using the code element, or as a block using the pre element. To automatically insert the html tags, use the u element. Wrapping the u element in a code element is not necessary. To make a closing tag, such as htmluse the data-end attribute. To make a self-closing tag, such as img, use the data-self attribute. These attributes to not need values.

/* some text wrapped in a pre tag, with a data-lang attribute to set the language in the top-right corner. */

SVG Icons

These flexible icons are pretty awesome, I'm just saying. In order to get the scale effect to work without the page flickering, I have to use scale3d(x, x, x) to these elements to enable hardware acceleration. Unfortunately, this makes the icons appear blurry sometimes. If anyone knows a solution, let me know! I fixed the blurry thing, but this is still a work in progress.
